# Grocery List App

A simple and elegant Grocery List app built with **React** and **Tailwind CSS**. Users can add, edit, and delete items from their list — all in a sleek dark UI!

##  Features

-  Add grocery items
-  Edit existing items
-  Delete items
-  Dark mode design
-  Built with React functional components and Hooks
